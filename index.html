hrn

Project hrn is short for Human Readable Number, a simple(< 1 Kb) javascript for browserjs / nodejs library to format number into human-readable string.


Install 

npm install hrn

Then import it. 

// import library use script tag.
<script type="text/javascript" src="dist/hrn.min.js"></script>

// or ES6 style.
var hrn = require('hrn');

1. Usage

hrn(1234000);                   // '1.2 M'
hrn(1234000, 3);                // '1.234 M'
hrn(1234000, 3, 'en');          // '1.234 M'
hrn(12340, 2, 'zh_CN');         // '1.23 ä¸‡', `en` / `zh_CN` supported, `en` is default.

2. Customize formatter

var formatter = ['kmgtpezy'.split(''), 1e3];

hrn('1234000', 1, formatter)    // '1.2 m'
